# Техническое задание: Сайт-презентация курса «Контуринг тела»

## Контекст

Создай сайт-презентацию обучающего курса для студентов школы моментального загара под брендом **li_zagar_tan**. Формат — слайдовая презентация в стиле премиальных лендингов: каждый раздел занимает один экран без вертикального скролла, переключение между слайдами через стрелки/клавиатуру/боковую навигацию.

**Контент курса:** в файле `course-content.md` (прикреплён). **Текст автора менять нельзя** — копировать дословно, только разносить по слайдам согласно структуре ниже.

**Картинку из исходного docx-файла** автора использовать как иллюстрацию (если будет приложен docx — извлеки из него image1, например через `unzip docx && ls word/media/`; если изображение приложено отдельно — используй его). Размещение картинки опиши ниже.

## Технологический стек

- **Next.js 15** (App Router) + **TypeScript**
- **Tailwind CSS** для стилей
- **Framer Motion** для переходов между слайдами и микроанимаций
- **Google Fonts:** Playfair Display (заголовки) + Inter (основной текст)
- Деплой: **Vercel**

## Дизайн-система

### Палитра

Пастельно-нюдовая база с брендовым акцентом. Используй CSS-переменные:

```css
:root {
  /* Основа — нюд/беж */
  --bg-base: #FAF6F1;        /* фон слайдов, тёплый молочный */
  --bg-card: #F4ECE2;        /* фон карточек, светлый беж */
  --bg-accent: #EDE0D0;      /* акцентные блоки, тёплый нюд */

  /* Текст */
  --text-primary: #2B2520;   /* тёмно-коричневый, не чёрный */
  --text-secondary: #6B5D52; /* приглушённый коричневый */
  --text-muted: #9B8B7E;     /* для подписей, мета-инфо */

  /* Брендовый акцент */
  --accent: #D5A14F;         /* основной брендовый */
  --accent-soft: #E5C28A;    /* осветлённый для hover/фонов */
  --accent-dark: #B8852E;    /* затемнённый для нажатий */

  /* Линии */
  --line: #E0D3C1;           /* разделители, рамки */
}
```

**Правило:** не используй чистый белый (#FFF) и чистый чёрный (#000). Все цвета — из палитры выше.

### Типографика

- **Заголовки слайдов:** Playfair Display, 64-80px на десктопе, 36-48px на мобиле, font-weight 400
- **Курсив-акцент в заголовках:** тот же Playfair Display, italic, цвет --accent. Пример: «Анализ *ценности*»
- **Подзаголовок-надпись над разделом** (Eyebrow): Inter, 12-14px, uppercase, letter-spacing 0.2em, цвет --text-muted. Пример: «Введение», «Физика», «Анатомия»
- **Основной текст:** Inter, 16-18px, line-height 1.6, цвет --text-primary
- **Подписи и мета:** Inter, 12-14px, цвет --text-muted

### Сетка и отступы

- Слайд занимает 100vh и 100vw, **без вертикального скролла**
- Контент центрируется в контейнере max-width 1200px
- Внутренние отступы слайда: `padding: 80px 120px` на десктопе, `padding: 40px 24px` на мобиле
- Между блоками текста — щедрые отступы (margin 32-48px)

### Карточки

```css
.card {
  background: var(--bg-card);
  border: 1px solid var(--line);
  border-radius: 4px; /* минимальный радиус, премиальный вид */
  padding: 32px;
}
```

Никаких теней (box-shadow) или градиентов — только плоский премиум-минимализм.

### Нумерация карточек

Когда нужны карточки 01/02/03 — номер выводи крупно (Playfair Display 32-40px, цвет --accent), над заголовком карточки.

## Навигация (вариант B из обсуждения)

1. **Стрелки слева/справа** внизу экрана (компонент `ArrowControls`)
2. **Клавиатура:** ←/→ переключение слайдов, Home — на первый, End — на последний
3. **Боковая навигация справа** (компонент `SideNav`):
   - Колонка точек, сгруппированных по 5 блокам курса
   - Между группами — тонкий разделитель + название блока (вертикальный текст или короткая метка справа)
   - Активная точка крупнее и в цвете --accent
   - При наведении на точку — всплывающая подсказка с названием слайда
4. **Индикатор № X / 19** в правом верхнем углу (компонент `SlideNumber`), Inter mono или Inter regular, --text-muted

**Анимация переходов:** Framer Motion, fade + slight slide (x: 30 → 0). Длительность 0.4-0.5s, easing easeOut.
